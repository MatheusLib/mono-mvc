# Monolito MVC (baseline)

Objetivo: implementar o dominio minimo usando MVC tradicional para servir de baseline.

Estrutura sugerida:
- `cmd/api` servidor HTTP
- `internal/controllers` handlers MVC
- `internal/models` entidades e persistencia
- `internal/services` regras de negocio

Foco: simplicidade e rapidez de entrega na Onda A.
